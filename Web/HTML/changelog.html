Release History

Newest first
  • v1.0.0 Current 09 Jul 2026
    • Console redesign — the "terminal ledger" theme, a staged boot sequence, and the multi-page site you're reading now.
    • Studio — a server-rendered analysis browser and run log alongside the live dashboard.
    • Consolidated backend — one FastAPI process serves the API, the dashboard, and every page.
  • v0.9.0 Jun 2026
    • Advanced tier — correlation and regression, wired through statsmodels.
    • Group-by — the medium and advanced tiers can split a column across a category.
  • v0.8.0 Jun 2026
    • Categorical tier — counts and proportions for label columns.
    • Dataset telemetry — live shape, analyzable/categorical split, complete vs. reduced counts.
  • v0.5.0 May 2026
    • Caching layer — results memoised on (tier, column, group); repeat questions are free.
    • Overview endpoint — dataset summary served without recomputation.
  • v0.3.0 Apr 2026
    • HTTP API — the engine wrapped in a FastAPI service with a JSON contract.
    • Static dashboard — the first browser front-end for picking a column and reading the result.
  • v0.1.0 Mar 2026
    • The engine — engine.py: basic descriptive statistics over a cleaned dataframe, and the missing-data rule that everything since has protected.
