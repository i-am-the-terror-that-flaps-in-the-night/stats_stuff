How These Were Taken

Scope & honesty

Figures are indicative, measured locally against the bundled sample dataset — 50 rows by 12 fields — on a single worker, the same free-tier shape the live demo runs on. They describe this dataset at this size; a larger file would move the first-call numbers, though the cached path stays flat by construction.

"Cached" is the honest common case: because results are memoised on (tier, column, group), anyone re-opening a figure — or a second visitor asking the same question — is served from memory. The point of publishing the cold numbers too is that nothing here is hidden behind the cache.
