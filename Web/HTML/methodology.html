The Pipeline

Ingest → Report

A request walks through four stages. Each stage is pure: the same CSV always produces the same numbers, so a result can be cached and trusted.

Stages, in order
StageDoesGuards against
1 · IngestRead the CSV into a dataframe.Missing file → a clean "dataset unavailable", never a crash.
2 · CleanTrim the frame, normalize blanks and null-like tokens.Stray whitespace or NA masquerading as data.
3 · CoerceCast each analyzed column to numbers; unreadable cells become missing.A single text cell silently poisoning a whole column.
4 · AnalyseRun the requested tier and return the figures.Recomputation — identical requests are served from cache.
02

The Four Tiers

Depth on demand

One column, four depths of question. The numeric tiers build on each other; the categorical tier answers a different kind of question entirely.

TierInputAnswers
basic1 numeric columnCentre & spread: count, mean, median, mode, min, max, std, variance.
mediumnumeric + optional groupEverything in basic, split across the categories of a grouping column.
advancednumeric + optional groupRelationships and significance — correlation and regression against other fields.
categorical1 label columnCounts and proportions for each distinct label.

The Formulas

8 figures

The base tier reports eight figures over the n values that survive coercion. Spread uses the sample estimator — dividing by n − 1, not n — because the data is a sample, not the whole population.

StatisticMeaningFormula
countValues that survived coercion.n
meanArithmetic average; pulled by outliers.(Σ xᵢ) / n
medianMiddle value once sorted; robust to outliers.50th percentile
modeMost frequent value(s); every tie is returned.argmax freq(x)
min / maxSmallest and largest observed values.min x · max x

04

Missing Data

Dropped, never invented

This is the rule the whole engine is built to protect.

A cell that can't be read as a number — a blank, NA, ?, or null — is dropped from that column's statistics, never replaced with a zero or an average. Filling a gap with a made-up number would quietly pull the mean, shrink the variance, and dress a hole up as a data point. So the engine refuses to.

The visible cost is honest: in the bundled dataset, five of the ten numeric columns each lose exactly one unreadable cell, so their figures summarize 49 values, not 50. The count is reported alongside every result precisely so that reduction is never hidden.

05

Determinism & Caching

Same in, same out

The dataframe is immutable for the life of the process, and every tier is a pure function of (tier, column, group). That means each answer is stable — so the engine memoises it. The first request for a figure computes it; every identical request after that is served straight from memory, no recomputation. The heavy one-time cost of the advanced tier's regression import is paid once, then never again.
