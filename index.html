        03

        Statistic Reference

        8 figures

        What each figure in the readout means — and how the numbers stay honest.

        count n
        How many values went into the figures — missing cells are dropped, so some columns show 49, not 50.
        mean
        Arithmetic average of the values. Sensitive to extremes — a single outlier pulls it.
        median
        The middle value once sorted (the 50th percentile). Barely moved by outliers.
        mode
        The most frequent value. Returned as a list, so every tie is shown — Age has four.
        min
        The smallest value observed.
        max
        The largest value observed.
        std sample · n−1
        Standard deviation: the typical distance of a value from the mean.
        variance = std²
        The standard deviation squared — the same spread in squared units.

        Missing data Cells that can't be read as a number — a blank, NA, ?, or null — are dropped, never filled in. Five of the ten columns each lose exactly one such cell, so their stats summarize 49 values instead of 50. Nothing is invented to close the gap.
